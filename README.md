##  1. O Problema Real
Estudantes de cursos de alta carga técnica, como **Engenharia de Software**, enfrentam frequentemente sobrecarga cognitiva e dificuldade em priorizar entregas complexas de cálculo e programação. A dispersão de prazos entre diferentes disciplinas gera ansiedade e desorganização acadêmica.

O **EduFlow Pro** atua diretamente nessa "dor", oferecendo uma interface centralizada que utiliza a psicologia das cores e sistemas de priorização visual para reduzir a fadiga mental e garantir que o foco seja mantido nas tarefas críticas.

##  2. Público-Alvo
* Estudantes universitários de tecnologia e áreas exatas.
* Alunos que necessitam de organização visual e categorização por disciplinas.
* Usuários que buscam interfaces modernas com alto contraste e foco em usabilidade.

##  3. Funcionalidades Principais
* **Gestão Dinâmica de Tarefas:** Criação, listagem e exclusão de atividades em tempo real.
* **Sistema de Priorização (Pin):** Permite fixar tarefas essenciais (como provas e entregas de projetos) no topo da lista.
* **Barra de Progresso em Tempo Real:** Visualização imediata da evolução das atividades concluídas.
* **Filtros de Estado:** Alternância rápida entre tarefas "Pendentes", "Concluídas" ou visualização total.
* **Acessibilidade e Design:** Interface baseada em princípios de **WCAG** (Web Content Accessibility Guidelines) para garantir contraste e legibilidade.

##  4. Tecnologias Utilizadas
* **Linguagem Principal:** JavaScript (ES6+ Vanilla).
* **Interface:** HTML5 Semântico e CSS3 (Variáveis nativas e Design Neon).
* **Testes Automatizados:** Jest (Ambiente JSDOM).
* **Qualidade de Código:** ESLint (Análise estática/Linting).
* **Automação (CI):** GitHub Actions para validação contínua de Build, Testes e Lint.

##  5. Instalação e Execução
1.  Clone este repositório:
    ```bash
    git clone [https://github.com/seu-usuario/eduflow-pro.git](https://github.com/seu-usuario/eduflow-pro.git)
    ```
2.  Navegue até a pasta do projeto:
    ```bash
    cd eduflow-pro
    ```
3.  Instale as dependências de desenvolvimento (necessário Node.js):
    ```bash
    npm install
    ```
4.  Para visualizar a aplicação acesse o Link do Site Publicado:
    ```bash
    https://jordana-code.github.io/EduFlow-Pro-Dashboard-Academico/
    ```

##  6. Testes e Qualidade
Para garantir a confiabilidade da aplicação, o projeto conta com rotinas de teste e análise estática:
* **Executar Testes Unitários:** `npm test`
* **Executar Análise de Lint:** `npm run lint`
